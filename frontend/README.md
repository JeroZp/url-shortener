# Frontend

React single-page application for the URL shortener project.

## Stack

- React 19
- Vite
- React Router
- Vitest and Testing Library

## Development

```bash
pnpm install
pnpm dev
```

## Build and Test

```bash
pnpm build
pnpm test
pnpm lint
```

## Notes

- The frontend is served behind the AWS Academy infrastructure described in the root README.
- Use the backend API exposed by the deployed environment or by the local Docker Compose stack.
