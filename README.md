# URL Shortener — Final CI/CD Project

URL shortener with click analytics. Academic project to demonstrate a complete CI/CD pipeline using AWS ECS Fargate, Terraform, and GitHub Actions.

## Stack

- **Backend:** FastAPI (Python 3.12)
- **Frontend:** React 18 + Vite
- **Data:** PostgreSQL 16 (RDS) + Redis 7 (ElastiCache)
- **Infrastructure:** Terraform + AWS ECS Fargate + ALB
- **Pipeline:** GitHub Actions
